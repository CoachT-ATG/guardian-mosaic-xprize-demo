# Contributing

This repository is a public contest demo only.

Rules:
- Do not add private Mosaic Organism internals.
- Do not add secrets, API keys, or PII.
- Keep prompts, logs, and screenshots sanitized.
- Prefer small, reviewable changes.

Suggested workflow:
1. Open an issue or draft note.
2. Update the public demo artifact.
3. Verify that the change remains contest-safe.
4. Keep the README and evidence files in sync.
