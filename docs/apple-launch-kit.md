# Apple Launch Kit

This kit collects the official Apple pages and the assets needed to finish enrollment, App Store Connect setup, and TestFlight distribution.

## Official Apple Pages

- Apple Developer Program enrollment: <https://developer.apple.com/programs/enroll/>
- Apple Developer Program overview: <https://developer.apple.com/programs/>
- App Store Connect workflow: <https://developer.apple.com/help/app-store-connect/get-started/app-store-connect-workflow>
- Add a new app record: <https://developer.apple.com/help/app-store-connect/create-an-app-record/add-a-new-app>
- TestFlight overview: <https://developer.apple.com/testflight/>
- App Store Connect overview: <https://developer.apple.com/app-store-connect/>

## What You Need To Do Manually

- Sign in with your Apple Account and complete the Apple Developer Program enrollment.
- Accept the developer agreement.
- Pay the Apple Developer Program membership fee if you are not using a waiver.
- Create an app record in App Store Connect.
- Add the app’s bundle identifier and signing configuration in Xcode.
- Upload the first build.
- Configure TestFlight internal or external testing.

## Assets To Prepare Before Enrollment

- legal entity name or individual legal name
- legal authority for the account holder
- Apple Account with two-factor authentication
- D-U-N-S Number if enrolling as an organization
- public website and support URL
- App Store name
- bundle identifier
- app subtitle
- keywords
- description
- privacy policy URL
- support contact email
- marketing screenshot set
- beta review notes
- TestFlight feedback email
