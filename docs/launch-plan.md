# Launch Plan

## Goal

Launch a contest-safe public demo and a sellable first-responder pilot with the smallest practical cash outlay.

## Operating Rules

- Build locally and in Workspace first.
- Keep the public repo sanitized and contest-safe.
- Sell one paid pilot before overbuilding.
- Use Google Cloud and Gemini only for the minimum deployed proof path.
- Enforce hard budget gates, quota limits, and deny-by-default behavior.

## Recommended Offer

`ATG Responder Coherence Operator`

Target buyers:
- law enforcement agencies
- fire / EMS departments
- dispatch teams
- academies
- peer-support and wellness units
- unions and municipal public-safety programs

Initial offer:
- 30-day pilot
- daily resilience check-ins
- weekly micro-training
- sponsor dashboard
- end-of-pilot summary report

Suggested pricing:
- individual responder: $97 to $197
- small peer-support cohort: $997 to $2,500
- department or academy pilot: $5,000 to $7,500

## Build Path

1. Local prototype
   - run the workflow locally
   - use mock data and JSONL receipts
   - keep private logic out of the public repo

2. Workspace surfaces
   - Google Docs for scripts and SOPs
   - Sheets for lead tracking, pilot roster, and revenue ledger
   - Forms for intake and feedback
   - AppSheet for the public pilot dashboard

3. Minimal XPRIZE proof path
   - one deployed app
   - one Google Cloud product
   - one Gemini API call
   - one dashboard screenshot
   - one usage log
   - one demo video under 3 minutes

## Budget Guardrails

- Default spend target before first revenue: $0 to $250
- Do not assume cloud budgets are hard caps; they are alerts only
- Keep API calls and deployment scope intentionally small
- Do not use LLMs for sensitive health data or clinical claims

## Evidence To Capture

- invoice or receipt
- user count
- pilot summary
- anonymized logs
- dashboard screenshots
- API usage record
- revenue ledger
- safety and privacy policy

## Immediate Next Step

Package the first small paid pilot and produce the public proof bundle:
- repo
- demo video
- pilot evidence
- revenue evidence
- minimal deployment record
